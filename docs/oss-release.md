# OSS Release Notes

## Positioning

Blog Agent Kit is a local article-generation workspace, not a CMS publisher.

## Default Non-Goals

- CMS publishing
- Social posting
- Email sending
- Private style profiles
- Private source scraping
- Hosted schedulers

## Release Checklist

- No private paths.
- No private names.
- No API keys.
- No generated article drafts from private topics.
- Tests pass.
- License is present.
